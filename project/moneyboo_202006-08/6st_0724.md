#### 1-0. 풀 리퀘스트
 - 한주동안 작업해온 결과물 풀 리퀘스트
 - 충돌이 일어나지 않고 성공 !
 
 #### 1-1. ID
 - 정보를 저장할때 index 보다 ID를 생성해서 넣어주자는 의견.
 - ex c20072501
 - 여기서 첫번째 c 는 카테고리를 의미 ( 어떤 정보가 담겨있는지 의미)
 - 두번째는 년도, 월, 일
 - 세번째는 몇번째인지 생성해서 저장하는 것인지 ( ex 01, 02, 03, ... )
 - 카테고리는 **c** 리스트는 **l**, 은행정보는 **b**
 - ID 생성함수는 utils/filter.js 에서 생성하여 이용할 것.
 
 #### 1-2. 쿠키 코드 정리
